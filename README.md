## 🎯 Descripción del Proyecto

**SmartLaunch** es un lanzador interactivo con diseño de **smartwatch** desarrollado para la materia **NTICX (Nuevas Tecnologías de la Información y la Conectividad)**. Este proyecto funciona como un centro de acceso centralizado que permite a los estudiantes navegar hacia diferentes recursos educativos sobre la evolución de las computadoras, historia de la tecnología y alfabetización digital.

El proyecto incluye una galería interactiva con **6 computadoras históricas** que abren ventanas emergentes con información detallada, una línea de tiempo completa hasta 2026, y videos educativos integrados de YouTube.

## 🚀 Funcionalidades

### ✨ Características Principales

- **Diseño Smartwatch** - Interfaz estilo reloj inteligente con reloj digital en tiempo real
- **Botones Lanzadores** - Acceso rápido a 3 secciones principales:
  - 📊 Línea de Tiempo Interactiva
  - 📖 Historia de las Computadoras
  - 💻 PC / Evolución Tecnológica
- **Galería Interactiva** - 6 cards con imágenes conceptuales y ventanas emergentes con información detallada
- **Actividad "Un poco de historia"** - Juego de arrastrar y soltar para relacionar hechos con computadoras
- **Línea de Tiempo Dinámica** - Hasta 2026 con generaciones y preguntas interactivas
- **Videos Educativos** - Integración de YouTube con reproducción embebida
- **Responsive Design** - Adaptable a dispositivos móviles, tablets y desktop

### 🎮 Actividad Interactiva

La actividad "Un poco de historia" permite a los estudiantes:

1. Leer los **10 hechos históricos** (A-J)
2. **Arrastrar** cada hecho hacia la computadora correcta
3. **Verificar respuestas** con retroalimentación inmediata
4. **Reiniciar** la actividad para practicar nuevamente

### 📊 Línea de Tiempo

La línea de tiempo incluye:

- **Desde 1623 hasta 2026** con hitos clave
- **6 Generaciones** de computadoras (1ª a 6ª)
- **Preguntas interactivas** que relacionan cada hito con historia mundial
- **Filtros por generación** para navegación rápida
